# 🗺️ MS-009 — Módulo SSMA

## Visão Geral
Módulo de Segurança, Saúde e Meio Ambiente para conformidade com NRs (NR-1, NR-4, NR-6, NR-7, NR-10, NR-35) e gestão de indicadores de segurança do trabalho.

## Entregas Planejadas

### Core
- [ ] Registro de acidentes/incidentes (CAT)
- [ ] Permissão de Trabalho (PT) — NR-10/NR-35
- [ ] DDS — Diálogo Diário de Segurança
- [ ] Controle de EPI (entrega/devolução/validade)
- [ ] Inspeções programadas
- [ ] ASO e exames periódicos (PCMSO)

### Indicadores
- [ ] LTIR (Lost Time Injury Rate)
- [ ] TFCA (Taxa de Frequência com Afastamento)
- [ ] TFSA (Taxa de Frequência sem Afastamento)
- [ ] Dias sem acidente por obra

### Integrações
- [ ] RH (dados de colaboradores, ASO)
- [ ] Estoque (EPI vinculado ao estoque)
- [ ] Frotas (checklists de segurança veicular)
