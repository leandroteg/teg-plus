# ✅ REQ-004 — Interface Mobile

## Descrição
Aprovadores devem conseguir visualizar e decidir sobre requisições através de um smartphone, sem precisar instalar nenhum app.

## Critérios de Aceite
- [x] Página `/aprovacao/:token` responsiva (mobile-first)
- [x] ApprovaAi `/aprovaai` otimizada para toque
- [x] Botões com área de toque adequada (≥ 44px)
- [x] Carregamento < 3s em 3G
- [x] Funcional sem login
