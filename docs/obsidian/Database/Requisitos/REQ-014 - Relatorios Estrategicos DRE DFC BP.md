# 📋 REQ-014 — Relatórios Estratégicos — DRE, DFC, BP, Fluxo de Caixa

## Descrição
Relatórios de nível estratégico para Controladoria e Diretoria, com dados consolidados gerados a partir dos lançamentos financeiros do Omie.

## Critérios de Aceite

### DRE — Demonstração do Resultado do Exercício
- [ ] DRE consolidado e por centro de custo / projeto
- [ ] Comparativo mensal e acumulado (YTD)
- [ ] Filtros por período, CC, projeto, natureza financeira

### DFC — Demonstração do Fluxo de Caixa
- [ ] DFC pelo método direto
- [ ] Exportação para Contabilidade

### BP — Balanço Patrimonial
- [ ] Dados de BP alimentados a partir dos lançamentos financeiros
- [ ] Sincronização com sistema contábil (Omie ou externo)

### Fluxo de Caixa Previsto
- [ ] Consolidação de CP + CR futuros em um único calendário de caixa
- [ ] Projeção por dia / semana / mês / trimestre
- [ ] Alertas de saldo insuficiente projetado

### Orçamento Previsto × Realizado
- [ ] Comparativo por centro de custo e projeto
- [ ] Desvios destacados com drill-down para lançamentos de origem
